Course Assignments

Assignment #1

You are to write a program using WebGL that will allow you to demonstrate twist and tessellation. The user should be able to enter both the level of tessellation---the number of subdivisions---and the angle θ. You can have the original vertices specified inside your program as well as both the level of tessellation---the number of subdivisions---and the angle θ. You can display your results as filled triangles or by their edges.
